# dix-mille

This is an app to enable players to count score at the dix-mille game. One also call this game [farkle](https://fr.wikipedia.org/wiki/Farkle).

I wanted to develop this app and make it available on the app store, but unfortunately the apple developer subscription is too expensive for a free app to be released :)
This app is then designed to be used on the web via phones.
